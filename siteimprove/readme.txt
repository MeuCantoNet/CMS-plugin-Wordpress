## Installation and configuration

> Siteimprove Plugin can be installed like any other Wordpress module. Place it in the plugins directory for your site and enable it on the `wp-admin/plugins.php` page.

> Visit the Siteimprove Plugin settings page and regenerate auth token if you need it.
